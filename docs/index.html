Quick Start Guide

Get up and running with DocsForge MCP in minutes

1

Install Dependencies

Clone the repository and install required packages

git clone https://github.com/NoManNayeem/docsforge-mcp.git
cd docsforge-mcp
npm install
2

Set Up Google Cloud

Enable APIs and download credentials

# Enable Google Docs API and Google Drive API
# Download credentials.json to project root
3

Build and Configure

Build the project and configure Claude Desktop

npm run build

# Add to claude_desktop_config.json:
{
  "mcpServers": {
    "docsforge": {
      "command": "node",
      "args": ["/path/to/docsforge-mcp/build/index.js"]
    }
  }
}
4

Start Using

Restart Claude Desktop and begin creating documents

"Create a new Google Doc with a title and some content"
