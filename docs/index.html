Quick Start

Get up and running in under 10 minutes

1

Clone & Install

Clone the repository and install dependencies

git clone https://github.com/NoManNayeem/google-docs-mcp-server.git
cd google-docs-mcp-server
npm install
2

Setup Google Cloud

Enable APIs and download credentials

# Enable Google Docs API and Google Drive API
# Download credentials.json to project root
3

Build & Configure

Build the project and configure MCP Client or Claude Desktop

npm run build

# Add to claude_desktop_config.json:
{
  "mcpServers": {
    "llm2docs": {
      "command": "node",
      "args": ["/path/to/build/index.js"]
    }
  }
}
4

Start Using

Restart Claude Desktop and begin creating documents
