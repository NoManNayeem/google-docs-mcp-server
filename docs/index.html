Quick Start
Get Running in Minutes

Simple setup process to get LLM2Docs working with Claude Desktop

1

Install Dependencies

Clone the repository and install required packages

Terminal 
git clone https://github.com/NoManNayeem/google-docs-mcp-server.git
cd google-docs-mcp-server
npm install
2

Set Up Google Cloud

Enable APIs and download credentials

Google Cloud Console 
# Enable Google Docs API and Google Drive API
# Download credentials.json to project root
3

Build and Configure

Build the project and configure Claude Desktop

Configuration 
npm run build

# Add to claude_desktop_config.json:
{
  "mcpServers": {
    "llm2docs": {
      "command": "node",
      "args": ["/path/to/google-docs-mcp-server/build/index.js"]
    }
  }
}
4

Start Using

Restart Claude Desktop and begin creating documents

Claude Desktop 
"Create a new Google Doc with a title and some content"
